# Movie Search App
## Academic purposes Movie Search App

## Description

### This app uses OMBD api to search through the OMDb Movie Database, you can search a movie by title and get a quick plot description and an IMDb rating. This App was made as a personal project to practice GitHub. 

## Stack 

[![Stack](https://skillicons.dev/icons?i=java,maven,spring,thymeleaf)](https://skillicons.dev)

## Setup 

### Clone the repostiory through git clone https://github.com/vladimir-maintser84/movie-search-app
### Optional: Get an API key from OMDb website and put it into MovieService
### Run the Application: mvn spring-boot:run OR run the main application class in your IDE
### Use the link localhost:8080/showForm in your web browser

## Endpoints

/showForm -  Search form
/processForm - - Process search results
